# Booking-System
A role-based booking system built in Python with a Tkinter GUI and MySQL backend. Includes secure login flow, user authentication, real-time data display, validation logic, and error-handling to ensure data integrity and system reliability. ## Features
- **Role-Based Access Control**  
  Separate interfaces for admins, instructors, and students.

- **Secure Login Flow**  
  User authentication with credential verification.

- **Graphical User Interface**  
  Built using CustomTkinter and `ttk.Treeview` for structured, interactive data display.

- **MySQL Backend**  
  Persistent data storage and reliable data retrieval.

- **Validation & Error Handling**  
  Ensures data integrity and prevents invalid input or system crashes.
